jekyll的安装和配置

安装

按照jekyll中文的教程进行安装

Jekyll Bootstrap

官网：http://jekyllbootstrap.com

GitHub：https://github.com/plusjade/jekyll-bootstrap/

git clone https://github.com/plusjade/jekyll-bootstrap.git
  • 把整个文件夹中的内容Copy到jekyll新建的blog目录（比如yushiro.github.io）
  • 修改_config.yml title: 博客网站名字 配置disqus帐号名 配置Google Analytics帐号
  • 修改_include\themes\twitter\post.html 替换为
  • 修改_includes\themes\twitter\default.html 增加如下代码，语法高亮的syntax.css
<link href="/css/syntax.css" rel="stylesheet">
  • 修改jekyll默认的markdown解析器为redcarpet（可以识别```为代码段）
markdown:    redcarpet
redcarpet:
  extensions: [fenced_code_blocks]
  • 如果不想使用”评论“功能
  comments :
    provider : false #disqus
  • 安装、切换theme样式
rake theme:install git="https://github.com/jekyllbootstrap/theme-the-program.git"
rake theme:switch name="the-program"
  • rake的安装
sudo gem install rake --no-rdoc --no-ri
