Javascript

JS中String类型的判断

var s = "asdf";
var strObj = new String("asf");
console.log(s instanceof String);   //返回的是false;
console.log(typeof s === "string"); //返回true;
console.log(strObj instanceof String); //返回true;

参考：http://stackoverflow.com/questions/203739/why-does-instanceof-return-false-for-some-literals

Node.js

NodeJS一些比较坑的地方

安装

  • 在win下安装完Nodejs后， 最好安装一下VS2010 C++，至少jsdom这个组件如果不安装VS2010 C++就会无法正常安装。
  • 在WIN下， 最好设置一下NODE_PATH环境， 指向node modules的安装目录，否则在运行js的时候，会提示模块找不到 NODE_PATH=C:\Users\yushiro\node_modules
